Detailed Description

High-level façade for interacting with ONFI-compliant NAND devices.

Wraps the low-level interface HAL with helpers for initialization, block erase/program/verify cycles, parameter queries, and common maintenance routines required by the Micron 3D NAND parts used in this project. All public members are exposed so tooling and sample applications can orchestrate self-tests or production flows without duplicating protocol logic.
